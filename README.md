# Scientific Calculator - Java Swing & AWT

Welcome to my Scientific Calculator project! This project is a powerful and versatile application built using Java Swing and AWT, designed to perform both basic mathematical operations and advanced calculations like trigonometric functions and logarithms.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Trigonometric functions: sine, cosine, tangent
- Logarithmic functions
- User-friendly graphical interface
- Object-Oriented Programming (OOP) principles applied for clean code

## Technologies Used

- Java
- Java Swing
- Java AWT

## Getting Started

### Prerequisites

- Java Development Kit (JDK) installed on your machine.
- A Java IDE (Eclipse, IntelliJ IDEA, etc.) for code editing.

### Installation

1. Clone the repository:
  
2. Open the project in your Java IDE.

## Usage

1. Run the application from your Java IDE.
2. The Scientific Calculator GUI will appear, allowing you to perform various calculations.
3. Choose the type of calculation you want to perform using the buttons provided.
4. Input the necessary values and click the relevant operation button.
5. The result will be displayed on the screen.
